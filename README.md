# 📸 Camera Shop Management System (Java GUI)

A desktop application built using **Java Swing/AWT** and **OOP principles** to manage a camera store's inventory and sales.

## ✨ Features
* **Inventory Management:** Add, Update, and Delete camera models.
* **User Interface:** Clean GUI for easy navigation.
* **Search Functionality:** Filter cameras by brand or price.
* **OOP Concepts:** Implemented Inheritance, Encapsulation, and Polymorphism.

## 🚀 How to Run
1. Clone the repository.
2. Open the project in **IntelliJ IDEA**, **Eclipse**, or **NetBeans**.
3. Run the `Main.java` file.

## 🛠️ Technologies Used
* **Language:** Java 
* **GUI Library:** Swing / JavaFX
* **Concepts:** Object-Oriented Programming (OOP)
